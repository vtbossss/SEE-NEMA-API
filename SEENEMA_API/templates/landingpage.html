To view the API documentation, please visit the following endpoint:

/api/schema/docs/

Explore the available endpoints and learn how to interact with the API.

Authentication

The SEE-NEMA API uses JWT (JSON Web Token) authentication. Before using the API, you need to register an account.

Register an Account

To register, make a POST request to the /api/register/ endpoint with your desired username and password:

POST /api/register/
Content-Type: application/json

{
  "username": "your_username",
  "password": "your_password",
  "email": "your_email@example.com"  // Optional, if you want to include an email
}

If registration is successful, you will receive a success message.

How to Authenticate

After registering, obtain a JWT token by making a POST request to the /api/auth/ endpoint with your username and password:

POST /api/auth/
Content-Type: application/json

{
  "username": "your_username",
  "password": "your_password"
}

If successful, you will receive a token in the response:

{
  "token": "your_jwt_token"
}

Include the token in the Authorization header for all authenticated requests:

Authorization: Bearer <your_token>

Example Request to Add a Movie

Here's how you can add a movie to your watchlist:

POST /api/watchlist/add/ 
Content-Type: application/json
Authorization: Bearer <your_token>

{
  "title": "Inception"
}
