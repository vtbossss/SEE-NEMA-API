What is SEE-NEMA API?

The SEE-NEMA API allows users to create and manage a personalized movie watchlist. It offers features like:

  • Adding movies to your watchlist
  • Updating or removing movies
  • Fetching movie details from the TMDb API when you add a movie title
  • Retrieving complete movie information like title, description, and release date
  • Secure authentication with JWT

How it Works

When you add a movie to your watchlist by providing a title, the SEE-NEMA API will automatically fetch all the movie details, such as the title, description, and release date, from the TMDb API and save it to your watchlist.
